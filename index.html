API Documentation

This is a Express API made to communicate to a .NET API

Endpoints

Index/Documentation

Endpoint: /

Method: GET

Example: http://localhost:7025/

Create User

Endpoint: /user/:name/:age

Method: POST

Parameters:

  • :name (string) - User's name
  • :age (number) - User's age

Example: http://localhost:7025/user/John/25

Retrieve All Users

Endpoint: /users

Method: GET

Returns: Array of user data

Example: http://localhost:7025/users

Retrieve Single User by ID

Endpoint: /user/:id

Method: GET

Parameters:

  • :id (string) - User's ID

Returns: User data for the specified ID

Example: http://localhost:7025/user/123

Update User by ID

Endpoint: /user/:id

Method: PUT

Parameters:

  • :id (string) - User's ID

Example: http://localhost:7025/user/123 in Postman:'{"name": "John", "age": 26}'

Delete User by ID

Endpoint: /user/:id

Method: DELETE

Parameters:

  • :id (string) - User's ID

Example: http://localhost:7025/user/123
